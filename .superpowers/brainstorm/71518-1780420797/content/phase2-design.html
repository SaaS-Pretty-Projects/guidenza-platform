Phase 2 — Deep Learning Experience

Quizzes, enhanced progress, polished certificates

Quiz Data Model

courses/{courseId}/modules/{moduleId}/quizzes/{quizId}
  ├── title: string
  ├── questions: [
  │     { id, questionText,
  │       options: [A,B,C,D],
  │       correctAnswer: number }
  │   ]
  ├── passingScore: number (70%)
  └── maxAttempts: number

Quiz Attempts (per user)

users/{uid}/quizAttempts/{attemptId}
  ├── courseId, moduleId, quizId
  ├── answers: [{ questionId, selectedAnswer, correct }]
  ├── score: number (%)
  ├── passed: boolean
  └── completedAt: Timestamp

Progress Upgrade

Module completion now requires: mark done + quiz passed. Course completion requires all modules done + all quizzes passed before certificate is issued.

What unlocks

  • Auto-graded quizzes for instructors
  • Meaningful progress (% by quiz score, not clickthrough)
  • Certificate only when mastery is demonstrated
